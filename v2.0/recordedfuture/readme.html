This app integrates threat intelligence from Recorded Future with Phantom Cyber.

You must have API access to Recorded Future to use this app. Use the test connectivity action to confirm your configuration.

The ip reputation action accepts both IPv4 and IPv4 addresses.

The file reputation action accepts file identity hashes using MD5, SHA1 or SHA-256 algorithms.

The domain reputation action accepts Domains and other DNS names.

The url reputation action accepts a URL (including 'http://' or 'https://')

The lookup vulnerability action accepts CVE vulnerability identifiers. It also accepts other advisory identifiers (see documentation for list.) Only CVE vulnerabilities return risk scores.
