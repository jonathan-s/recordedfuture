Recorded Future's app for Splunk Phantom allows clients to work smarter, respond faster, and strengthen their defenses through automation and orchestration. The app provides a number of actions that enables the creation of Playbooks to do automated Enrichment, Correlation, Threat hunting and Alert handling.

CHANGELOG %RELEASE%

New Actions

alert rule id lookup
Searches for Alert Rules matching the search string and returns the name and id of each match.
alert data lookup
Return detailed information of any triggered Alerts for the provided Alert id and timerange.
url reputation
Returns risk and timing information for an URL.
ip intelligence
Returns detailed intelligence for an IP address.
domain intelligence
Returns detailed intelligence for a domain.
file intelligence
Returns detailed intelligence for a file (a hash).
vulnerability intelligence
Returns detailed intelligence for an vulnerability (ex a CVE).
url intelligence
Returns detailed intelligence for an URL.
